# HelloMyTeam 소개
회사에 가면 매일 아침 하는 인사관리 시스템을 나의 애정있는 나의 팀에 적용해보자는 취지에서 만들어졌습니다.

## 이러한 기능들을 제공할 것 입니다.
### V1.0
- 구글로그인
- 팀 생성 / 팀 가입
- 팀원 수락 / 팀원 스탯 성장실
- 자유게시판 / 공지 / 전략실
- 팀 홍보 게시판

### V2.0
- 투표 기능
- 투표 기능 기반 캘린더 
- 투표 인원에 따른 포메이션 생성 페이지
- 투표 인원에 따른 스탯 평가 


# 기술 스택

BackEnd
- Language: Java 8, ES6,   
- Framework: Springboot: 2.6.7
- DB: Maria DB
- CI/CD: git action 

FrontEnd
- 작성예정
